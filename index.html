Introduction

AlgorythmHelper Library is designed to simplify working with algorithms and data structures. It enables quick implementation of popular algorithms and manipulation of data structures, optimizing the development process.

Features:

  • Easy-to-use sorting, searching, and graph algorithms.
  • Convenient API for data structures (arrays, queues, stacks, trees).
  • Optimized implementations for fast execution.

Installation

Install via Python Package Index (pip):

pip install algorythmhelper
Algorithms

  1. Searching

    Binary Search

    Used to find an element in a sorted list.

    from algorythmhelper.search import binary_search
    arr = [1, 3, 5, 7, 9]
    target = 5
    result = binary_search(arr, target)
    print(result) # Output: 2
    • Input: sorted list arr, target element target.
    • Output: index of the element or -1 if not found.

    Linear Search

    Sequentially iterates through the list to find the target element.

    from algorythmhelper.search import linear_search
    arr = [4, 2, 8, 6]
    target = 8
    result = linear_search(arr, target)
    print(result) # Output: 2

  2. Sorting

    QuickSort

    Sorts a list using the "divide and conquer" method.

    from algorythmhelper.sorting import quick_sort
    arr = [3, 1, 4, 1, 5]
    result = quick_sort(arr)
    print(result) # Output: [1, 1, 3, 4, 5]

    MergeSort

    from algorythmhelper.sorting import merge_sort
    arr = [10, -1, 2, 5]
    result = merge_sort(arr)
    print(result) # Output: [-1, 2, 5, 10]

  3. Graph Algorithms

    Breadth-First Search (BFS)

    from algorythmhelper.graphs import bfs
    graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F'],
    'D': ['B'],
    'E': ['B', 'F'],
    'F': ['C', 'E']
    }
    start = 'A'
    result = bfs(graph, start)
    print(result) # Output: ['A', 'B', 'C', 'D', 'E', 'F']

    Depth-First Search (DFS)

    from algorythmhelper.graphs import dfs
    result = dfs(graph, start)
    print(result) # Output: ['A', 'B', 'D', 'E', 'F', 'C']
Data Structures

Queue

from algorythmhelper.structures import Queue
queue = Queue()
queue.enqueue(1)
queue.enqueue(2)
print(queue.dequeue()) # Output: 1
print(queue.is_empty()) # Output: False

Stack

from algorythmhelper.structures import Stack
stack = Stack()
stack.push(10)
stack.push(20)
print(stack.pop()) # Output: 20
print(stack.peek()) # Output: 10

Binary Tree

from algorythmhelper.structures import BinaryTree
tree = BinaryTree()
tree.insert(5)
tree.insert(3)
tree.insert(7)
print(tree.inorder_traversal()) # Output: [3, 5, 7]
